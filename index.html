Thoughts on Progressive Enhancement

Progressive enhancement is the practive of building a website in which that core content is presentable for all users. After the core content is available and presentable then you add advanced features. This is super important because people using the website use all diffrent kinds of things like computers or there phone or say an ipad and using this approach allows even people with slow internet to view the core functions or information on the website without having to use the advanced parts https://developer.mozilla.org/en-US/docs/Glossary/Progressive_Enhancement
